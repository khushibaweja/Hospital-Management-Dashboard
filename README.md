# Hospital-Management-Dashboard
Interactive Excel dashboard analyzing hospital performance using Pivot Tables, Pivot Charts, KPIs, Slicers, and data visualization techniques.


**Hospital Performance Dashboard (Excel)**

** Project Overview**
This project is an interactive Hospital Performance Dashboard built using Microsoft Excel. The dashboard analyzes hospital operations and provides insights into patient trends, revenue, appointment performance, and patient satisfaction through KPIs, Pivot Tables, Pivot Charts, and Slicers.

 **Objectives**
- Analyze hospital performance using Excel.
- Track revenue and patient metrics.
- Monitor appointment status and no-show trends.
- Evaluate patient satisfaction.
- Enable interactive filtering using slicers.

**Tools & Features**
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- KPIs
- Slicers
- Conditional Formatting
- Data Cleaning
- Excel Formulas

   **Key Performance Indicators (KPIs)**
- Total Revenue
- Total Patients
- Average Patient Satisfaction Score
- Patients with No Missed History
- Frequent Missers (>3 Missed Appointments)
- No Show Rate

** Dashboard Includes**
- Department-wise Revenue Analysis
- Department-wise Patient Count
- Appointment Status Analysis
- Patient Satisfaction Analysis
- Previous Missed Appointment Analysis
- Interactive Slicers (Department, City, Age Group, Appointment Status)

 **Dataset Information**
The dataset contains 10,000 patient records with the following fields:
- Patient ID
- Age & Age Group
- Gender
- City
- Department
- Appointment Type
- Appointment Status
- Insurance
- Waiting Time
- Chronic Disease
- Patient Satisfaction Score
- Total Bill
- Revenue Details
  
** Key Insights**
- Compared revenue generated by different hospital departments.
- Identified departments with the highest patient volume.
- Analyzed appointment completion, cancellation, and no-show trends.
- Studied the relationship between previous missed appointments and patient behavior.
- Evaluated patient satisfaction across different categories.
